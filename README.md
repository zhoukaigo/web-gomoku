# 网页五子棋游戏

一个基于HTML5 Canvas的网页五子棋游戏，支持双人对战。

## 功能特点

- 🎮 完整的五子棋游戏逻辑
- 🎨 美观的棋盘和棋子设计
- 📱 响应式设计，支持移动设备
- 🔄 一键重新开始游戏
- 🏆 自动判断胜负

## 游戏规则

1. 黑子先行，双方轮流落子
2. 在横向、纵向或斜向连成五子者获胜
3. 点击棋盘交叉点放置棋子

## 技术栈

- HTML5
- CSS3 (渐变、阴影、动画)
- JavaScript (ES6+)
- Canvas API

## 快速开始

1. 克隆或下载项目文件
2. 用浏览器打开 `index.html`
3. 开始游戏！

## 文件结构

```
web-gomoku/

├── style.css       # 样式文件
```
